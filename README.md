# java-board 프로젝트

이 프로젝트는 Java 기반의 게시판(보드) 웹 애플리케이션입니다.

## 주요 폴더 구조

```
board-demo/   # 주요 게시판 백엔드(스프링 등)
demo/         # 샘플/테스트 앱
```

## 실행 방법

### 1. Gradle 빌드

```bash
./gradlew build
```

### 2. 실행 (예시)

```bash
cd board-demo
./gradlew bootRun
```

## 개발 환경 및 기술 스택
- Java
- Spring Framework
- Gradle
- 기타: Docker, Git, 등
